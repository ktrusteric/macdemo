## 🎯 核心技术架构

### 前端技术栈
- **框架**: React 18 + TypeScript
- **状态管理**: Redux Toolkit
- **UI组件**: Ant Design
- **路由**: React Router v6
- **HTTP请求**: Axios
- **样式**: Tailwind CSS

### 后端技术栈
- **框架**: FastAPI (Python)
- **数据库**: MongoDB
- **认证**: JWT Token
- **API文档**: OpenAPI/Swagger
- **AI集成**: ai.wiseocean.cn

### 数据流架构
```
用户行为 → 前端组件 → Redux状态 → API服务 → 后端处理 → 数据库 → AI分析 → 个性化推荐
```
